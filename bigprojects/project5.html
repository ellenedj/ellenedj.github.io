Building visualisation

Type of project

Group project (4 people)

Project goal

Visualize data from the Avans building so that students can interact with the product and find a comfortable classroom or workspace to study in. Avans can use the feedback data that students give to the rooms to improve the quality of classrooms and workspaces.

Tasks

Generating ideas for the group, designing different user interfaces (UI). Ask other students for feedback on the UI. Improving the UI based on user experience (UE).

Description

The end product is an interface that changes based on data. Based on the temperature and how many people are in the room the puppets in the building and the colour of the classroom/workspace changes. This way, when students arrive in the Avans building, they can see where and what workspaces are busy or can choose a room with a comfortable temperature. Avans can also use this data to improve the workspaces based on the the feedback of the students and data of how busy certain work spaces are.Ultimately Avans can manage the energy levels of the building by using the data of each room.
